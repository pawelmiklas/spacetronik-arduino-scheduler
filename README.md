# Spacetronik SPE-223HB Desk Scheduler

Automatic height control for a Spacetronik SPE-223HB desk using an Arduino Nano, DS3231 RTC, and a 2-channel 5V low-level relay module. The desk raises or lowers at predefined times and will auto-raise on boot if power is restored during an active UP period. Runs Monday to Friday only.

## Hardware

- Arduino Nano
- DS3231 RTC
- 2-channel 5V low-level relay module (10A)
